# 📊 Sales Data Analysis Project

## Overview
A complete end-to-end sales analysis project built entirely in **Microsoft Excel**.  
The project analyzes over $2.29M in sales data across 5,009 orders, 793 customers, and 4 years (2014–2017).

---

## 🎯 Project Objectives
- Analyze overall financial performance (sales, net sales, COGS, returns, discounts)
- Identify top-performing product categories, regions, states, and cities
- Uncover sub-category and time-based sales trends
- Build an interactive dashboard for dynamic filtering and decision-making

## 🔧 Tools & Techniques Used
- **Power Query** — data cleaning and transformation
- **Pivot Tables** — multi-dimensional analysis across 6 dimensions
- **Interactive Dashboard** — slicers for City, Category, Salesperson, and Returns
- **Data Modeling** — 4 connected tables: Orders, Returns, Shipping Cost, People
- **Data Visualization** — bar charts, pie charts, line charts, map-style tables

## 📌 Key Findings

| Metric | Value |
|--------|-------|
| Total Sales | $2,297,201 |
| Net Sales | $2,116,697 |
| Total Orders | 5,009 |
| Total Customers | 793 |
| COGS | $1,555,845 |
| Total Discounts | $297,687 |
| Total Returns | $180,504 |

**Top Category:** Technology — $836,154  
**Top Region:** West — $725,458  
**Top City:** New York City — $256,368  
**Top State:** California — $457,688

---

## 💡 Critical Insight
Despite strong revenue ($2.29M), profitability is under pressure from high COGS and discounts.  
→ **Priority:** Margin optimization over pure revenue growth.
